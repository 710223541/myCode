规划畅想：

  1. 设计上要突出年轻活力，春天，风和日丽，春意盎然，构化融入农场元素（要求可以动）
  2. 设计春夏秋冬四种主题的样式，样式表可以随着时间算法自动切换

技术要求：

  1. 基于html5/css3不考虑IE8以下低版本浏览器兼容
  2. 交互部分完全使用js开发，基于seajs进行模块化
  3. php处理后台数据，nodejs处理前台ajax请求，AngularJs处理在前台处理后台数据数据
  4. 响应式适应各种终端
  5. UI设计标准化，THML5化，各种icon/button/字体/导航条/布局等要求全站通用

交互要求：

  1. css3动画、阴影、背景、边框
  2. js+css3动画

前台模块：

  1. 首页
  2. 作品
  3. 博客
  4. 关于

后台模块：

  1. 首页
  2. 用户管理，删除，权限
  3. 模块管理>文件管理
  4. 访问量统计、反问IP地点、返回时间+停留时间
